2. How to Sign In

  1. Go to the application URL (provided separately once deployed on Vercel).
  2. Enter one of the email/password combinations above.
  3. Click Sign In. New team members can click Sign up to self-register (they will default to the Employee role).

3. What Admins Can Do

  • Everything an Employee can do, plus:
  • Access the Admin section in the sidebar (Employees only see modules relevant to them).
  • View total users, admin count, employees on record, and open job postings at a glance.
  • Promote any employee to Admin, or demote/remove them (except the two fixed admin accounts).
  • Post job openings, run payroll, approve leave/expense/regularization requests.
  • Manage company Settings — departments, leave policies, holidays, roles, integrations, billing.

4. What Employees Can Do

  • View their dashboard with a personalized greeting.
  • Apply for leave, submit expense claims, raise helpdesk tickets.
  • View attendance, payroll, onboarding checklist, and performance goals.
  • Cannot access the Admin Dashboard or change other users' roles.

5. Module Walkthrough

Dashboard — Overview, KPIs, pending approvals, announcements
Employees — Directory, org chart, employee profiles
Attendance — Daily attendance, mark attendance, regularizations
Leave — Apply, approve/reject, balances, holidays
Payroll — Run payroll, payslips, salary cost by department
Recruitment — Post jobs, candidate pipeline
Onboarding — New hire checklists with task tracking
Performance — Goals/OKRs, performance reviews
Expenses — Submit claims, approve/reject, category breakdown
Assets — Add assets, assign/reassign to employees
Helpdesk — Raise and track IT/HR support tickets
Reports — Workforce analytics, exportable reports
Settings — Company profile, departments, policies, database
Admin Dashboard — User & role management (Admins only)

6. Suggested Demo Flow for the Client

  1. Sign in as Admin (sumanthbolla97@gmail.com) — show the Admin Dashboard and user management.

  3. Approve a pending leave request and expense claim from the Dashboard.
